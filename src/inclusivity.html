Inclusivity

I am neurodivergent, and I am very welcoming if you are too. We can set things up to make sure you feel as comfortable and safe as possible. For example:

  • Sitting side-by-side so we don’t make eye contact;
  • Learning English in a quiet place with dim lights;
  • Taking regular breaks;
  • Repeating information if you find it hard to concentrate;
  • Having any comfort items you’d like with you (soft toys, blankets, favourite books etc.).
Learning English can be fun and exciting for everyone including you!
